# ⚽ Football Analytics Portfolio  
### Data-driven scouting, performance analysis, and decision-support models

Hi 👋

I'm Antonio Pennacchia, a Data Analytics & Business Intelligence professional with over 5 years of experience, with a strong passion for football analytics.

This repository collects the football-related analytics projects that best represent how I approach data in the sports industry.

My focus is not limited to pure performance analysis: I’m particularly interested in transforming data into actionable insights for scouting departments, technical staff, and decision-makers.

The projects included here range from:

- **Applied case studies** developed on real football datasets  
- **Performance analysis models**  
- **Player profiling and segmentation projects**  
- **Experimental analytics concepts**
